# Trabalho do Ginásio - Módulo 5

Este é o nosso programa para gerir os sócios do ginásio Fitness+. Fizemos isto para a disciplina de Programação (PSI).

### Quem fez o trabalho:
* Martim Pereira
* Martim Bernardo

### O que o programa faz:
O programa serve para guardar os dados das pessoas que andam no ginásio. No menu principal dá para:
- Registar novos sócios (pede o nome, idade, morada, etc).
- Ver a lista de toda a gente que já está inscrita.
- Procurar por um sócio específico para ver os detalhes.
- Mudar os dados se alguém se enganar a escrever.
- Apagar um sócio da lista.

### Notas importantes:
- O limite máximo é de 100 sócios.
- Usámos structs para organizar a informação toda.
- O código está preparado para mostrar bem os acentos na consola do Windows.
