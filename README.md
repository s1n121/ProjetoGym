# Projeto Ginásio 

Este é o nosso programa para gerir os sócios do ginásio Fitness+.

### Quem fez o trabalho:
* Martim Pereira
* Martim Bernardo

### O que o programa faz:
O programa serve para guardar os dados dos sócios que andam no ginásio. No menu principal dá para:
- Registar novos sócios (pede o nome, idade, morada, etc).
- Ver a lista de todos os sócios que estão inscritos.
- Procurar por um sócio específico para ver os detalhes.
- Mudar os dados de algum sócio.
- Apagar um sócio da lista.

### Notas importantes:
- O limite máximo é de 100 sócios.
- Usámos structs para organizar a informação toda.
- Temos validação em tudo para termos a certeza que a pessoa escreve o tipo de dados certo e o programa não dá erro.
